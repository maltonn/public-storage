# Ikasama Roulette
イカサマのできるルーレットです。次のように使います。  
  
①「ジュースおごるルーレットしようぜ」などと誘い、ルーレットを回す  
↓  
②「どっちがいい？」などと、相手に色を聞く。（紫or緑）  
↓  
③ストップボタンを押させる  
↓
④相手が言ったことを音声認識して、相手の選んでいないほうでルーレットが止まる。
